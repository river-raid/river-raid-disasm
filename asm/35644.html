8B3C: Render an object with collision detection.
Two-phase rendering pipeline: (1) erase old sprite at 8B0A using XOR blending, (2) draw new sprite at 8B0C using OR blending, checking for pixel collisions during the draw phase. Sprite frame is selected from E's lower bits for sub-pixel alignment.
Input
A Sprite width in tiles (1-3)
BC Sprite frame size in bytes
D Sprite height in pixels
E Screen attributes (lower bits also select animation frame)
HL Pointer to sprite data array (base of all frames)
