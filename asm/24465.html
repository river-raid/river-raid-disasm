5F91: Main gameplay loop
Central game loop that runs continuously during active gameplay. Orchestrates all game subsystems in a fixed sequence each frame.
  • Input: Check Enter key (Caps+Enter → restart, Symbol+Enter → control select)
  • Timing: Increment tick counter (5EEF)
  • Render: Explosions, plane/terrain, viewport objects
  • Missiles: Two-pass player missile (erase then draw)
  • Projectiles: Tank shells, helicopter missiles
  • Scroll: Advance game world (66D0)
  • Fuel: Consume fuel (6DFF)
  • Controls: Dispatch to input handler based on 5F67
The loop is infinite. It terminates only via: game over (fuel empty, collision) or player death (jumps to death handler). Pressing Enter calls 6BBF (Caps+Enter → 5D35 to restart gameplay; Symbol+Enter → 6BD2 to return to control selection). The H key pause is handled entirely by the interrupt handler (6BDB) and does not exit the loop.
