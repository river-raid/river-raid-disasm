6256: Handle COLLISION_MODE_FUEL_DEPOT (initiate refueling)
Used by the routine at 6136.
Called from the collision dispatcher when COLLISION_MODE_FUEL_DEPOT is detected. Sets gameplay mode to GAMEPLAY_MODE_REFUEL and uses the plane's coordinates (instead of missile coordinates) for collision detection.
This allows the plane itself to interact with fuel depots via 61BB.
6256 LD A,$06 Set gameplay mode to GAMEPLAY_MODE_REFUEL in 5F68.
6258 LD ($5F68),A
625B LD B,$80 Load plane coordinates (Y=$80, X from 5F72) into 5EF3.
625D LD A,($5F72)
6260 LD C,A
6261 LD ($5EF3),BC
6265 JP $61BB Check for collision with fuel depot.
