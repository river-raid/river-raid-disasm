694D: Advance to next bridge/level
Called when all 64 terrain fragments of the current bridge are completed. Resets Y-position and advances to the next bridge. Bridge index wraps from 49 back to 1, but the player's progress counter in 5F6A continues incrementing for the wraparound calculation in 68E9.
Output
A Always set to 0
694D LD DE,$0000 Reset Y-position to start of new bridge.
6950 LD ($5F70),DE
6954 LD A,($5EF0) Increment bridge index at 5EF0.
6957 INC A
6958 LD ($5EF0),A
695B CP $31 If index reaches $31 (49), wrap to 1.
695D JP Z,$6963
6960 LD A,$00
6962 RET
