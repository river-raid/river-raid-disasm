703E: Load tank screen attributes
Returns tank attributes: $00 normally, $04 if tank is on river bank (bit 5 of D set).
Input
D Object definition byte (bit 5 = tank on bank flag).
Output
E Attributes value $00 or $04.
703E LD E,$00 Load E=$00. If D bit 5 set (tank on bank), load E=$04 instead.
7040 BIT 5,D
7042 RET Z
7043 LD E,$04
7045 RET
