6EC8: Process and render all active explosions
Iterates through the explosions set at 5F2E via pointer 5F62. Each explosion has 6 animation frames before being removed. Adjusts Y position based on scroll speed.
  • Entry format: [X_pos, Y_offset, frame_counter] where frame_counter bits 0-6 = frame (1-6), bit 7 = erase flag
  • Frame 1,5: small explosion (6F63)
  • Frame 2,4: medium explosion (6F67)
  • Frame 3: large explosion (6F6B)
  • Frame 6: erase explosion (6F6F) then remove entry
