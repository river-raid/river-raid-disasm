6136: Collision detection dispatcher
Central collision handler called during sprite rendering (via 8C45) when pixel overlap is detected. Saves registers, reads collision mode from 5EF5, and dispatches to the appropriate handler.
Mode Handler Description
COLLISION_MODE_NONE ($00) 8C3B Rendering only
COLLISION_MODE_FUEL_DEPOT ($01) 6256 Fuel depot refuel
COLLISION_MODE_MISSILE ($02) 61BB Missile hit
COLLISION_MODE_FIGHTER ($03) 615E Fighter hit
COLLISION_MODE_HELICOPTER_MISSILE ($04) 7415 Enemy missile
6136 POP HL Save registers: return address to 5F85, DE to 5F87, BC to 5F89.

613A LD ($5F87),DE
613E LD ($5F89),BC
6142 LD A,($5EF5) Load collision mode and dispatch to handler (see table above).
