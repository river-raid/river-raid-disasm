75A2: Advance right-facing ship or helicopter
Moves a right-facing ship/helicopter 2 pixels right, checking for terrain collision ahead.
Input
B Y position
C X position
D Object definition
75A2 PUSH BC Save BC, calculate terrain check position: C += $20 pixels.
75A3 LD A,C
75A4 ADD A,$20
75A6 LD C,A
75A7 CALL $8A4E Get terrain byte at (C, B). Load result into A.
75AA LD A,(HL)
75AB POP BC Restore BC. If terrain != 0, reverse direction.
75AC CP $00
75AE CALL NZ,$75D0
75B1 LD ($8B0A),BC Store position to 8B0A. Advance X position right by 2 pixels (INC C twice).
75B5 INC C
75B6 INC C
75B7 JP $7128 Continue to 7128 for rendering.
