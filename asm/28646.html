6FE6: Load pointer to right-facing enemy sprites
Returns pointer to sprite array for enemies facing right. Used by ld_enemy_sprites to select sprite direction based on enemy movement.
Output
HL Pointer to right-facing enemy sprite array at sprite_enemies_right.
ld_enemy_sprites_right 6FE6 LD HL,$8793 Load HL with sprite_enemies_right (right-facing sprites base address).
6FE9 RET
