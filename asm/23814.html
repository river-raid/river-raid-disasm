5D06: Return to control selection dialog
Used by the routines at 6BD2 and 6D17.
This entry point is used when returning to the control selection dialog from the game (via 6BD2) or from the overview mode. It switches back to IM 1, then calls clear_and_setup to display the control selection dialog.
After the user selects controls and game mode, execution continues at 5D10.
5D06 LD A,$3F Restore standard IM 1 interrupt vector table.
5D08 LD I,A Set the I register to $3F for IM 1 mode.
5D0A IM 1 Set interrupt mode 1.
5D0C EI Enable interrupts.
5D0D CALL $7804 Display the control selection dialog.
