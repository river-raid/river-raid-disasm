6BED: Process control state flags and trigger sound effects
Sound processing dispatcher called from interrupt handler. Reads 6BB0 control flags and calls appropriate sound routines. Multiple sounds can trigger simultaneously (e.g., fire + low fuel).
Bit Flag Sound Handler
0 FIRE_SOUND 8A02 (fire)
1 SPEED_NOT_FAST 6C5D (normal speed)
2 SPEED_CHANGED 6CB8 (fast speed)
1+2 Both speed bits 6CD6 (slow speed)
3 LOW_FUEL 6CF4 (warning)
4 BONUS_LIFE 6C31 (jingle)
5 EXPLODING 6C7B (explosion)
Flags are set by game logic (input handlers, collision, fuel system) and cleared by sound routines when complete.
6BED LD A,($5C08) Skip if LAST_K is 'h'.
6BF0 CP $68
6BF2 JP Z,$6C24
6BF5 LD HL,$6BB0 Check FIRE_SOUND → 8A02.
6BF8 BIT 0,(HL)
6BFA CALL NZ,$8A02
6BFD BIT 4,(HL) Check BONUS_LIFE → 6C31.
6BFF CALL NZ,$6C31
6C02 LD HL,$6BB0
6C05 BIT 5,(HL) Check EXPLODING → 6C7B.
6C07 CALL NZ,$6C7B
6C0A LD HL,$6BB0
6C0D BIT 3,(HL) Check LOW_FUEL → 6CF4.
6C0F JP NZ,$6CF4
6C12 LD A,(HL)
6C13 AND $06 Mask speed bits.
6C15 CP $02 Normal speed → 6C5D.
6C17 JP Z,$6C5D
6C1A CP $04 Fast speed → 6CB8.
6C1C JP Z,$6CB8
6C1F CP $06 Slow speed → 6CD6.
6C21 JP Z,$6CD6
