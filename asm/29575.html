7387: Invert shell coordinate delta
Subtracts E×4 from C for left-facing tank shell trajectory adjustment.
Input
C Current X position
E Delta value
Output
C Adjusted X position
invert_shell_coordinate_delta 7387 LD A,C C = C - E×4 (SUB E four times).
