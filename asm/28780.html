706C: Render balloon
Renders a balloon enemy sprite at the specified X position. Balloons are 2-tile wide animated objects that float above the river. Unlike other enemies, balloons disable collision detection during rendering.
  • Disables collision mode via 5EF5 (allows plane to pass through during render)
  • Animated sprite with $20 byte frames at 8972
  • Dimensions: 2 tiles wide (16px) × 16 pixels tall
  • Attributes: $0E (PAPER RIVER, INK BALLOON)
Input
E X position of balloon
706C LD B,$00 BC = (0, E): set X position in BC for object entry.
706E LD C,E
706F LD HL,$8972 Load balloon sprite 8972, set COLLISION_MODE_NONE to 5EF5.
7072 LD A,$00
7074 LD ($5EF5),A
7077 PUSH HL Push sprite addr, add balloon to 5F00 objects set, pop and call 6FEA.
7078 LD HL,$5F00
707B CALL $6EAB
707E POP HL
707F CALL $6FEA
7082 LD BC,$0020 Render sprite: frame size=$20, width=2, height=16px, attributes=$0E.
7085 LD A,$02
7087 LD DE,$100E
708A CALL $8B1E
708D RET
